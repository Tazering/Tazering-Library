# ***YAML Files***

- way of representing data

```yaml
Fruit: Apple
Vegetable: Carrot
Liquid: Water
Meat: Chicken
```
- Key-Value Pair
  - need space before value
  
```yaml
Fruits:
-   Orange
-   Apple
-   Banana

Vegetables:
-   Carrot
-   Cauliflower
-   Tomato 
```
- Arrays and Lists
  - same indent size after `-`

```yaml
Banana:
    Calories: 105
    Fat: 0.4 g
    Carbs: 27g

Grapes:
    Calories: 62
    Fat: 0.3 g
    Carbs: 16 g
```
- dictionary/maps
  - equal number of blank spaces
